# 武器特效优化计划（2026-07-23）

> 技术美术视角 · 原生 Canvas 2D · 程序化像素美术 + 代码效果层
> 目标：补齐 5 个武器的「视觉可信度」短板，全部走低风险改造（零或极少新素材，不引入 shadowBlur 每帧每物）。

## 设计支柱
1. **辨识度优先**：每个武器在屏幕上必须一眼可辨，不与其它武器/领域撞脸。
2. **动效服务打击感**：挥鞭、落雷、抛瓶都要有「发生→高潮→消散」的时序。
3. **性能红线**：辉光/落雷用 `globalCompositeOperation='lighter'` + 瞬时 `shadowBlur`，不在满屏每物每帧开 shadowBlur。

## 改造清单

### T1 血之飞刃 → 忍者飞刀 + 扇形发射
- `gen_assets.py::gen_blade()` 重写为 kunai：朝右(+x)尖刺刀身 + 握柄环 + 镂孔 + 缠绳，体积光。
- `weapons.js::fire()` blade 分支：以最近敌人为中心，`count` 片按 `~0.16rad` 间隔扇形抛出（之前是各自锁定不同敌人，方向发散）。

### T2 亡灵光环 → 红环 + 六芒星
- `gen_assets.py::gen_weapon_aura()` 重写为红色六芒星图标（升级卡缩略图一致）。
- `weapons.js::render()` aura 分支：由紫圈改为「红色圆环(脉冲) + 内嵌缓慢旋转六芒星(两等边三角形)」，与圣水蓝领域彻底区分。

### T3 噬魂长鞭 → 挥鞭动效
- `weapons.js::applyWhip()` slash 加 `bow` 随机弯曲方向。
- `render()` slashes 分支：由 `fillRect` 光带改为锥形曲线鞭（二次贝塞尔采样，柄粗→尖细）+ 末端裂响爆闪，带挥出/淡出；`maxLife` 0.18→0.22。

### T4 雷霆审判 → 天降落雷
- `weapons.js::strikeLightning()` 链首前插入天空起点（`sky` 标记，`y-360`）。
- `render()` bolts 分支：首点 `sky` 则从天绘制到首敌，落点画 `lighter` 爆闪。`stormcall`/`tempest` 神器共用。

### T5 圣水洗礼 → 抛物线投掷
- `weapons.js` 新增 `vials` 数组（constructor/reset/update）。
- `fire()` holywater 改为抛飞瓶（参数 `t`：线性位移 + `sin(πt)` 弧高），落地生成原圣水领域池。
- `render()` 绘制飞瓶（复用 `weapon_holywater` sprite，翻滚旋转）。`devour` 神器跟随领域不变。
